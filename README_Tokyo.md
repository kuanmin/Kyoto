### 🚂 荒川電車散步
- 都電荒川線（Tokyo Sakura Tram）  
- 東京市區內僅存的兩條路面電車之一。它不僅是一條交通路線，更是一條串連老東京「下町」風情的時光隧道。   
- 復古路面電車體驗：荒川線維持單節車廂運行，最高時速僅約 40 公里。它在部分路段與一般汽車共用道路，這種穿梭於巷弄、河堤與民宅間的低速旅行，讓遊客能近距離觀察當地居民的日常生活。  
- 東京櫻花路面電車（Tokyo Sakura Tram）：這是該線在 2017 年獲得的暱稱。沿線有多處賞花名所，特別是春季的櫻花與春、秋兩季的玫瑰花（沿軌道種植），是著名的花卉觀賞線路。  
- 深厚的下町庶民氣氛：路線連接了「三之輪橋」與「早稻田」，沿途經過許多充滿懷舊感的老社區、傳統商店街（如庚申塚站的巢鴨地藏通商店街）與古老的神社寺廟。  
- 推薦停靠  
  - 庚申塚站  
  - 巢鴨地藏通商店街（老人原宿）  
### 🥁 池袋最大祭典 池袋ふくろ祭り 
- 「袋祭」始於1968年，是由池袋站西口的4個商店會發起並舉辦的祭典活動，今年將迎來第58屆。在神轎遊行中，將會有16座大神轎和數千人的抬手齊聚在池袋站西口附近。熱鬧的吆喝聲將響徹這座城市，喚醒觀眾的節日激動之心。
- 在池袋西口公園還將舉辦歌謠表演和舞蹈表演，將有數家本地餐飲店在此出攤。遊客可一邊享受美食一邊感受節日氣氛。
- 17:00、神轎（mikoshi）遊行、太鼓、小吃攤
