  - 品種多，櫻花層次感超豐富
  - 平野神社の「櫻花祭」（Oukasai）是京都歷史最悠久的春季盛事之一。
    - 夜間點燈（夜櫻）期間： 2026年3月28日（六）至 4月12日（日）

    - 費用： 境內免費參訪；櫻花季期間進入「櫻苑」需付費（2026年預計為 500日圓）。  
    - 櫻花特色： 境內擁有約60種珍稀櫻花。從3月底開花的「魁櫻」到4月中旬的「平野妹背」，花期極長，即使在3月底前往也能欣賞到綻放的早櫻。

## 📅 4/6（星期一）｜京都鐵道＋古都風華列車＋大阪夜櫻  
- 🍳 早餐到 10:00（最晚 11:00 退房）

### 🚄 京都鐵道博物館  
- 京都鐵道博物館 [點我看影片](https://www.youtube.com/watch?v=MJaP6MEvVCM)  
  - 京都鐵道博物館，是日本最大規模的鐵道博物館。該館以「觀看、觸摸、體驗」為主題，展示了從蒸汽火車到新幹線等共 53 輛 具備歷史意義的退役列車，被譽為「活著的鐵道博物館」。   
  - 壯觀的扇形車庫與蒸汽火車: 梅小路扇形車庫：這座興建於 1914 年的車庫是日本國家重要文化財，保存了日本最大規模的蒸汽機車群。SL Steam 號體驗：遊客可以支付額外費用（約 300-400 日圓）搭乘由真實蒸汽機車牽引的客車，體驗往返約 1 公里的鐵道旅行，感受蒸汽火車的鳴笛與震動。檢修車庫觀摩：在「SL 第 2 檢修車庫」，遊客可以透過窗戶近距離觀察技術人員維護蒸汽火車的真實場景。   
  - 互動式體驗與模擬設施: 行駛模擬裝置：提供與職業司機訓練相同的模擬裝置，讓遊客穿上制服體驗駕駛火車的感覺（需事先購票或抽籤）。鐵道模型比例模型（Diorama）：館內擁有日本最大規模之一的電動模型場景，精細展示了鐵路運行的日常。安全技術展示：設有許多互動教具，讓遊客了解鐵路信號、隧道建設及自動檢票機的工作原理。   
  - 獨特的觀景與用餐空間: 天空露台（Sky Terrace）：位於本館 3 樓，可以同時俯瞰東海道新幹線、JR 京都線及山陰本線的列車頻繁通過，是鐵道攝影的最佳位置。主題餐廳與鐵道便當：2 樓餐廳可邊用餐邊觀賞窗外飛馳的列車。此外，遊客還可以在由舊車廂改建的餐飲區購買並享用特色「鐵道便當」。   
  - 2026年期間限定活動: 2月21日至5月6日舉辦大型鐵道玩具展。

### 🚃 乘坐古都風華特色列車 前往 大阪  
- 近鐵觀光特急 [點我看影片](https://www.youtube.com/watch?v=OyyLAVZJ3Gc)  
  - 官網 [點我打開網頁](https://www.kintetsu.co.jp/foreign/chinese-han/aoniyoshi/)  
  - 「AONIYOSHI」（青丹吉 / あをによし）於 2022 年 4 月開航，是一款連結大阪、奈良、京都三地的豪華觀光列車。其名稱「Aoniyoshi」源自於古代對奈良城的讚美枕詞，象徵著奈良古都的優雅與美麗。   
  - 列車特色與設計  
    - 列車以「1300年前的天平時代」為主題，斥資 3.3 億日幣打造，展現濃厚的奈良文化底蘊。   
    - 外觀： 車身採用象徵高貴與古代官位的「紫冠位」深紫色塗裝，輔以金色的天平文樣裝飾。  
    - 內裝： 內部壁紙與地毯點綴著來自正倉院寶物的「天平文樣」花鳥圖案，營造出如行走藝術品般的奢華氛圍。  
  - 布落格說明 [點我打開網頁](https://nicklee.tw/2936/kintetsu-aoniyoshi/)
