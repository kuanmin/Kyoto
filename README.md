### 鴨川  
- 鴨川賞櫻🌸[點我看影片](https://youtu.be/IAVA4uxrxsk?si=MxVohz-3KDM6NFJ0)  
  - 京都鴨川的跳腳石（飛び石）是橫跨河川的趣味造型踏腳石，以「烏龜」造型最著名，是熱門打卡點。從三條走到四條的河岸散策，特色在於結合現代與傳統的河畔風情，夏季可見「納涼床」，兩岸種滿櫻花，充滿悠閒的都市慢活氛圍，適合悠閒散步。   
  - 鴨川跳腳石（跳烏龜）：最著名的跳腳石位於鴨川與高野川交匯的「鴨川三角洲」（出町柳附近），這裡除了烏龜形狀，還有鳥、船等造型，供遊客跳著穿越河道。  
  - 三條大橋至四條大橋散策路線：這段路線是京都最具代表性的河畔景色之一。傳統與現代交織： 西側鄰近先斗町、木屋町，沿岸傳統町家與現代風格並存，充滿濃濃京都風味。自然美景與人文氣息： 河岸兩側栽種大量櫻花樹，春季是櫻花隧道，平常則是極佳的慢跑、散步道。晚上橋樑和餐廳燈火通明，營造出迷人的夜景。建議傍晚時分沿著河岸散步，從三條走到四條，可以同時感受夕陽下的河景與夜晚的浪漫氛圍。

### 晚餐。
   
### 平安神宮 櫻音夜。
- 京都平安神宮以其宏偉的朱紅色社殿與名勝庭園「神苑」內的約300棵櫻花聞名，特別是4月上旬至中旬盛開的「八重紅枝垂櫻」，宛如紅色雲彩般垂落，景象震撼。庭園內的池水倒影加上夜間點燈與音樂會，營造出優雅的夢幻氛圍，是京都著名的晚櫻賞櫻名所。   
- 南神苑的紅色花海： 從入口處的南神苑就能看到大量八重紅枝垂櫻，花瓣偏桃紅色，盛開時如同紅色天蓋般罩頂，震撼力十足。  
- 東神苑的棲鳳池倒影： 櫻花映照在棲鳳池水面上的倒影，搭配尚美館、泰平閣等建築，是經典的拍照場景。  
- 「細雪」的櫻花： 日本大文豪谷崎潤一郎在其名著《細雪》中，曾讚嘆平安神宮紅枝垂櫻之美。  
- 晚開的櫻花： 由於神苑以枝垂櫻為主，花期較染井吉野櫻稍晚，適合「櫻花季後半」來訪的旅客。  
- 櫻花音夜會與點燈： 每年4月上旬會舉辦夜櫻音樂會，在幽美音樂中賞夜櫻，體驗與白天不同的華麗氛圍。  
- 有報名4月3日晚上19:55音樂會。可以提早入場先參觀，時間到了再去音樂會的地點。（有預約~別遲到囉！）

### 🚄 京都鐵道博物館  
- 京都鐵道博物館 [點我看影片](https://www.youtube.com/watch?v=MJaP6MEvVCM)  
  - 京都鐵道博物館，是日本最大規模的鐵道博物館。該館以「觀看、觸摸、體驗」為主題，展示了從蒸汽火車到新幹線等共 53 輛 具備歷史意義的退役列車，被譽為「活著的鐵道博物館」。   
  - 壯觀的梅小路扇形車庫與蒸汽火車，興建於 1914 年的日本國家重要文化財，保存了日本最大規模的蒸汽機車群。   
  - 天空露台（Sky Terrace）可俯瞰東海道新幹線、JR 京都線及山陰本線的列車頻繁通過，是鐵道攝影的最佳位置，並享用特色「鐵道便當」。   
  - 2026年期間限定活動: 2月21日至5月6日舉辦大型鐵道玩具展。
